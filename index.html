⏳ กำลังเชื่อมต่อ Firebase... (หากใช้ไม่ได้ กด "ใช้งานแบบ Local" ด้านล่าง)
1. ไปที่ console.firebase.google.com
สร้าง Project ใหม่ → เลือก Firestore Database → Start in test mode
2. ไปที่ Project Settings → Add App → เลือก Web (</>)
Copy firebaseConfig มาวางด้านล่าง
3. วาง config ในช่องด้านล่างแล้วกด Save
รูปแบบ: { apiKey: "...", authDomain: "...", projectId: "..." }
⚠️ ใช้งานบน GitHub Pages?
ต้องเพิ่ม domain ใน Firebase Console ก่อน:
Authentication → Settings → Authorized domains
→ Add domain: ชื่อuser.github.io
